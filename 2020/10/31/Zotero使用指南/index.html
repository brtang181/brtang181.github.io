Zotero使用指南

孔子云:工欲善其事,必先利其器.对于一名21世纪的科研人来说,一款合适的文献管理软件能够提高工作效率.Zotero就为我们提供了一个很好的选择.便捷的操作与强大的功能,使之成为了辅助我们科研的”神兵利器”.本文旨在提供一个Zotero的简明使用指南,以便读者快速上手.关于Zotero的使用,最权威且全面的自然是Zotero的官方文档,但如果你在诸多细节中找不到重点,那么本文不失为一个很好的选择.本文的最后提供其他学习资源的链接,以供参考.现在,就让我们开始了解Zotero的文献管理之道吧.

初识Zotero

Zotero是一款免费,开源的文献管理软件,其名称”Zotero”由阿尔巴尼亚语”zotëroj”派生而来,有”掌握,占有”之意.

在Zotero中.每个文献构成一个条目(item),条目的类型多种多样,从书籍,网页,文档,到期刊文章,会议论文等等,一应俱全.依赖于其类型,每个条目中有不同的元数据(metadata).

如图所示,条目显示于Zotero的中央面板,元数据显示于右边面板.左边面板的上方是文件夹(collections),下方是标签(tags).每个条目可以附加若干附件(attachment),附件可以是注记(notes),文件(files)或是链接(links)等等.

下载与安装

像安装其他许多软件一样,请认准 Zotero官方网站,访问下载页面进行下载并安装.你需要同时安装浏览器插件Zotero Connector,我们很快会看到,这将极大便利我们添加条目(item)的过程.

添加条目与文件

如前所述,条目可以说是Zotero文献管理的”基本单元”,我们这里介绍添加条目的几种方式.

通过网络浏览器添加条目(推荐)

添加PDFs和其他文件

如果你需要添加本地的PDF或其他文件,只需将其拖入Zotero中,Zotero会自动尝试从中抓取元数据并创建条目.当然,你也可以将其拖入已有的条目使之成为该条目的附件.如果Zotero不能识别该文件,你也许不得不手动添加条目而后将文件拖入该条目中.独立于条目的文件是被允许的,不过这样的文件将不能享受Zotero的许多便捷功能.

手动添加条目

Zotero的设计旨在使我们尽可能少地手动添加条目,因此这应当是”最后一种办法”.点击绿色加号标志,选择你想要添加的条目类型并添加条目,并在右边面板输入文献信息即可.

同步

Zotero的同步可以使你通过网络访问你的Zotero library. Zotero同步分为数据同步与文件同步两种.

数据同步

数据同步可以使你将本地的条目,注记,标签等等(除了文件附件之外的一切)同步到Zotero服务器.为了使用这一功能,你需要创建一个Zotero账户.打开Zotero preference中的同步面板并输入登录信息,Zotero就会自动为你同步数据.当然你也选择可以关闭自动同步功能.数据同步是免费的且没有存储上限.

文件同步

如前所述,数据同步并不能够同步附件文件.Zotero提供了不同的文件同步方案,你可以使用Zotero File Storage或是使用WebDAV. Zotero File Storage为每位用户提供了300 MB的免费存储空间,更多的存储需要购买.详情可参考Zotero官网.

Better BibTex

Better BibTex(BBT)是Zotero的一款扩展程序(extension),旨在帮助我们更好的管理文献数据(bibliographic data).对于使用Latex或是Markdown等工具的创作者来说,BBT尤为便捷.

下载与安装

相比Zotero本身,Better BibTex的安装过程较为复杂,请参照官方安装指南进行安装.

BibTex基础

Latex中常使用BibTex来生成参考文献.要使用BibTex处理参考文献列表,首先需要准备好参考文献数据库.
